# Fungi Hackaton

Стартовый каркас проекта:

```text
.
├── client/         # Webpack + Phaser 3 клиент на чистом JS
├── server/         # Node.js сервер
├── libs/
│   ├── phaser/     # локальные расширения Phaser
│   └── spine/      # Spine runtime / adapters
└── package.json    # корень monorepo
```

Основные команды:

```bash
npm install
npm run dev:client
npm run dev:client:open
npm run dev:server
```
